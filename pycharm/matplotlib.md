## Pycharm matplotlib in interactive mode
## Solution:
```python3
import matplotlib as mpl
import matplotlib.pyplot as plt
mpl.use('macosx')

plt.plot(range(10))
```
+ or 
```
Disable Settings | Tools | Python Scientific | Show plots in tool window. I
```
